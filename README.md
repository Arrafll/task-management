# Task Management System  
A project & task management application built with **Laravel 11**.  
This application provides features for managing projects, tasks, progress tracking, overdue detection, and visual analytics.

---

## Features

### **Project Management**
- Create, edit, update, and remove projects.
- Track project progress based on task completion.
- Project status classifications (Planning, On Progress, Done).
- Automatic detection for problematic projects (overdue tasks + low progress).

### **Task Management**
- Assign tasks under projects.
- Manage task status (Planning, On Progress, Done).
- Track priorities and deadlines.
- Auto-detection of overdue tasks.
- Quick status change menu with dropdown submenu.

### **Dashboard & Analytics**
- Progress bar for each project.
- Monthly task completion chart (Highcharts).
- Pie chart for task distribution by status.
- Widgets for total tasks, done tasks, overdue tasks, and in-progress tasks.
